Help

Home Screen

Here, the user is able to toggle on/off the ability to show subtitles on the current screen. Using the buttons at the top, the user can select the profile screen, settings screen, open the transcibe studio screen, or this help screen.

Profile Screen

Here, the user is able to see statistics such as how many videos they have subtitled in total.

Settings Screen

Here, the user is able to toggle their prefences for the extension including the ability to customize how the subtitles are displayed on the screen.

Transcibe Studio

Here, the user is able to contribute and transcibe the audio from videos for other users to benefit from.
